## üîß Migration Steps

### **Step 1: Enable Maintenance Mode (5 min)**

**1.1. Enable Laravel Maintenance Mode**
```bash
cd /path/to/jurnal_mu
php artisan down --retry=60 --secret="v11-deploy-2026"
```

**1.2. Verify Maintenance Page**
- Visit homepage ‚Üí Should show "Service Unavailable" (503)
- Bypass URL: `http://yoursite.com/v11-deploy-2026` (admin access)

**1.3. Stop Background Workers (if any)**
```bash
supervisorctl stop laravel-worker:*
```

---

### **Step 2: Deploy Code (10 min)**

**2.1. Pull Latest Code**
```bash
cd /path/to/jurnal_mu

# Stash any local changes (should be none)
git stash

# Fetch and checkout v1.1.0
git fetch origin
git checkout v1.1.0

# Verify correct version
git describe --tags
# Output: v1.1.0
```

**2.2. Update Dependencies**
```bash
# Backend dependencies
composer install --no-dev --optimize-autoloader --no-interaction

# Frontend dependencies
npm ci

# Build frontend assets
npm run build
npm run build:ssr
```

**2.3. Verify Build**
```bash
# Check if public/build/ contains new assets
ls -lh public/build/

# Check manifest.json exists
cat public/build/manifest.json | head
```

## üîô Rollback Plan

### **Rollback Steps (15 min)**

**1. Enable Maintenance Mode**
```bash
php artisan down --retry=60 --secret="rollback-v10"
```

**2. Restore Database**
```bash
# Stop all processes using the database
mysql -u root -p -e "SHOW PROCESSLIST;"
mysql -u root -p -e "KILL [process_id];"  # If necessary

# Drop v1.1 database (CAREFUL!)
mysql -u root -p
DROP DATABASE jurnal_mu;
CREATE DATABASE jurnal_mu CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
exit

# Restore v1.0 backup
gunzip < jurnal_mu_v1.0_backup_YYYYMMDD_HHMMSS.sql.gz | mysql -u root -p jurnal_mu

# Verify restore
mysql -u root -p jurnal_mu -e "SHOW TABLES;"
```

**3. Rollback Code**
```bash
cd /path/to/jurnal_mu

# Checkout v1.0
git fetch origin
git checkout v1.0.0

# Reinstall v1.0 dependencies
composer install --no-dev
npm ci
npm run build
```

**4. Clear Caches**
```bash
php artisan optimize:clear
php artisan config:cache
php artisan route:cache
```

**5. Verify v1.0 Works**

## üêõ Known Issues & Workarounds

### **Issue 1: File Upload Max Size**

**Symptom:** Users cannot upload files > 2MB (PHP default)

**Workaround:**
```bash
# Edit php.ini
upload_max_filesize = 10M
post_max_size = 12M
```

**Restart PHP-FPM:**
```bash
sudo systemctl restart php8.2-fpm
```

---

### **Issue 2: Email Notifications Not Sending**

**Symptom:** Coaching assignments don't trigger emails

**Workaround:**
- Check `.env` mail configuration
- Verify queue is running: `php artisan queue:work`
- Test email manually: `php artisan tinker` ‚Üí `Mail::raw('Test', fn($msg) => $msg->to('test@example.com')->subject('Test'));`

---

### **Issue 3: Missing Translations**

**Symptom:** Some v1.1 features show English text instead of Bahasa Indonesia

**Workaround:**
- Check `resources/lang/id/` for missing keys
- Add translations for new features
- Clear cache: `php artisan cache:clear`
